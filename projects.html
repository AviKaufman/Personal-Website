Projects

This project aims to develop a neutrino interaction simulation for use with machine learning reconstruction algorithms, particularly in the context of liquid argon time projection chambers (TPCs) like those to be employed in the Deep Underground Neutrino Experiment (DUNE).

In many particle physics experiments, the raw data obtained from detectors is often ambiguous and requires complex reconstruction into tracks and interactions. However, the graphical representation of the data in tomographic slices presents an opportunity to apply machine learning techniques to facilitate quicker and more accurate reconstruction. Neutrino interactions in liquid argon TPCs are ideal candidates for such reconstruction techniques.

The Standard Model of particle physics predicts that when a neutrino interacts with a quark in a proton, a weak boson is exchanged, and a lepton (such as an electron, muon, or tau) or one or more hadrons (such as pions) can be detected. The energy spectrum of these particles can be modeled based on the energy of the incoming neutrino, which is a key factor in reconstructing the neutrino energy.

To accomplish the goal of using machine learning algorithms for neutrino event reconstruction, a simulation of a neutrino interaction will be developed. The simulation will produce a track or shower of simulated particles using a simple detector description. This simulated data will serve as a basis for testing reconstruction algorithms, particularly those utilizing machine learning techniques.
